Tech
Pure HTML + CSS + vanilla JS — no frameworks, no build step, no dependencies. One file: index.html.

Fonts — Syne (headings) + DM Sans (body) via Google Fonts
Figma embeds — iframes pointing to Figma's embed API
Routing — JS-driven page switching (no server needed)
Hosting — GitHub Pages (free)


How to update the site
Every update is just two steps:

Get a new index.html (make changes locally or ask Claude to generate one)
Go to this repo on GitHub → click index.html → click the pencil icon → paste the new content → click Commit changes

GitHub Pages deploys automatically in ~60 seconds.

Local preview
No build step needed. Just open the file in a browser:
bashopen index.html
# or on Windows
start index.html
